# Sales Data Warehouse Project

This project is a **Sales Data Warehouse** being developed in **Databricks**.  

## Purpose

The project is being developed for the **Hogarth Dhaka assessment** to demonstrate skills in data engineering, analytics, and working with cloud-based data platforms.

## Project Overview

- Data sources: Sales and transactional data
- Platform: Databricks
- Objectives: Data ingestion, transformation, and analysis to support reporting and business insights

## Getting Started

1. Clone this repository into your Databricks workspace using **Databricks Repos**.
2. Access and run the notebooks to explore the datasets and transformations.
3. Make sure you have the necessary permissions and access tokens for Databricks integration.
